ISO GPG Signature KaOS Signing Key

The November version ships with Plasma 6.2.3 as the default desktop and has a size of 3.6 GiB.

  • KDE Applications 24.08.3 are included, as are Linux kernel 6.11.9 (Linux-next is in the repo’s, following the 6.12 series), GCC 14.2.0, Glibc 2.40, Systemd 253.27, Kmod 33, Xorg-server 21.1.14, Qt 6.8.0, Mesa 24.2.7, Calligra 4.0.1, LibreOffice 24.8.3, Elisa and the latest versions of Falkon, Octopi, & Neochat.
  • For those wondering about gtk apps, Firefox 132, Chrome 133, Thunderbird 132, GIMP 2.99.18, Ardour 8.10.0 are among the available in their latest version.
  • Media Creation. To create reliable Media for booting the Live ISO, please read Create Install Media and follow the instructions for either using a USB flash drive or DVD disc. Though UEFI booting is supported, Secure Boot is not at this time. See this link for more on Secure Boot.
  • Authenticity check. It is advised to check the integrity of the downloaded ISO by comparing SHA256Sum and to check the authenticity through the signature of the ISO.
  • Known issues. Installing on RAID is currently not possible.
$ SHA256SUM KaOS 2024.11 b113a8acf19e382775b3d01cc9343d3c8025dc74c3284366833afe4b4e10bb13

Using GnuPG to verify the ISO signature

Besides checking the integrity of the downloaded ISO, it is advised to also check the authenticity through the signature of the ISO. For this download the KaOS Signing Key. Add this key to your keyring:

$ gpg --import KaOS.asc

Next download the ISO GPG Signature to the same directory as where you downloaded the ISO and verify with (adjust for correct year and date):

$ gpg --verify KaOS-2024.11-x86_64.iso.sig
